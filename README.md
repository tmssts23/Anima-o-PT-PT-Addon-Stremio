# Addon Stremio Animacao PT-PT

Projeto standalone para filmes e series de animacao em PT-PT.

## Fonte

- Filmes: `https://tvparapobreskids.com/filmes/`
- Series: `https://tvparapobreskids.com/series/`

## Inclui

- Catalogo de filmes de animacao
- Catalogo de series de animacao
- Meta e streams
- Sem catalogo de novelas

## Como correr

1. `npm install`
2. `npm run build`
3. `npm start`

Manifest:

- `http://127.0.0.1:7000/manifest.json`
